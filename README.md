<details>
<summary><b>🔗 Deploy to VPS</b></summary>
<br>


```console
$ git clone https://github.com/KenKaneki979/TianaBot
$ cd TianaBot
$ pip3 install -U -r requirements.txt
$ cp sample.env .env
```
> Edit .env with your values and then start bot with
```console
$ bash start
```
